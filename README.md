argo
====

## Dependencies

```sh
$> modprobe cdc-acm
$> pacman -S avrdude
$> go get -u -v github.com/hybridgroup/gort
$> gort arduino upload firmata /dev/ttyACM0
```

## Installation

```sh
$> go get -u -v github.com/sbinet/argo/...
```
